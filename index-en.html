Multiple running environment

The cross platform characteristic based on C language can adapt to complex host environment. More is comming.

P2P Networking

Each agent can be connected to other agent and eventually form a P2P topology network. Depending on this network, the administrator can operate any node in the network at any time.

Connection

Each agent can be either a server, a client or both through TCP interconnection.

No dependence && Small

Full static compilation can be used directly without considering the third party system library. The products are all around 200K, which is small enough.

Built-in shell

Upload files / Download files / socks service / port forwarding / reverse port forwarding, etc. All of the above instructions can be sent through the built-in shell in admin.

Regular maintenance

More utilities are under development. You're welcome to try them out and report the bugs. I'll grow with it.
